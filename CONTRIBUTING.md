# Contributing to FifaEvent

Thank you for your interest in contributing to FifaEvent!

## Local Setup

1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate the environment and install dependencies: `pip install -r requirements.txt`
4. Run the app: `flask run`

## Pull Request Guidelines

1. Ensure all tests pass.
2. Code must be formatted with `black` and pass `flake8` linting.
3. Keep PRs small and focused on a single issue or feature.
4. Add a description of your changes to the CHANGELOG.

## Reporting Issues

If you find a bug, please create an issue with detailed steps to reproduce it.
